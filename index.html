Welcome to my First Project

I've done my best to present a smart design for PaperCo's move into a second building. Below are a few additional notes.

You can view the code on Github

***Make sure to click the tiny "notepad icons" within the chart for more information***

• Equipment room will contain modem, firewall (physical device), router, DMZ servers, Honeypot, Application servers, RADIUS/AAA server, and a switch to lead to workstations and the connecting fiberoptic cable.

• Depending on where exactly the ISP installs the modem and the location of building number 2, Cat6 cables are best to provide high speed and bandwidth. I recommend using Cat6 cabling for the upstream devices
